박스 그림(Box Plot)

Seaborn 사용하기

  • 히스토그램과 같은 연속적인 분포를 곡선화해서 그린 그림

카운트그림(Count Plot)

  • 범주형 column의 빈도수를 시각화 -> Groupby 후의 도수를 하는 것과 동일한 효과

  • 숫자형 변수와 하나 이상의 범주형 변수의 관계를 보여주는 함수

  • scatter plot과 유사하게 데이터의 수치를 표현하는 그래프

히트맵(Heatmap)

  • 데이터의 행렬을 색상으로 표현해주는 그래프

Numpy 시작하기

python 모듈인 numpy를 알아보고 선형대수 연산 기초를 알아본다.

numpy 모듈 사용하기

실행 시켜보면 넘파이를 사용한 배열 이 더 빠른것을 확인 할 수 있다.

다차원 배열을 선언할 수 있고 shape 을 이용해 배열의 형태를 확인할 수도 있다.

numpy 연산

  • 벡터와 스칼라 연산
  • 벡터와 벡터 연산
  • 각 벡터의 요소 끼리 연산

array의 indexing과 slicing

  • 파이썬의 리스트와 유사하게 진행

Array의 Broadcasting

  • 피연산자가 연산 가능하도록 변환을 하여 연산을 진행하는 것을 Broadcasting 이라 함.

  • M x N, M x 1

  • M x N, 1 x N
  • M x 1, 1 x N

영벡터(행렬)

  • 원소가 모두 0인 벡터(행렬)

일벡터(행렬)

  • 원소가 모두 1인 벡터(행렬)

대각행렬(diagonal matrix)

  • Main diagonal을 제외한 성분이 0인 행렬

항등행렬(identity matrix)

  • Main diagonal이 1인 대각행렬
  • 인자로 들어가는 dtype의 기본 값은 float

행렬곱(dot product)

  • 행렬간의 곱연산

트레이스(trace)

  • main diagonal의 합

행렬식(determinant)

  • 행렬을 대표하는 값들 중 하나

역행렬(inverse matrix)

  • 행렬 A에 대해 AB = BA = I 를 만족하는 행렬 B = A^-1

고유값과 고유벡터(eigenvalue and eigenvector)

  • 정방행렬 A에 대해 $Ax = \lambda x$를 만족하는 상수$\lambda$와 이에 대응하는 벡터
  • 1차원 벡터의 norm 값을 구하는 함수 작성
  • 2차원 벡터의 singular matrix 인지 확인하는 함수 작성

Mat

  • opencv에서 사용되는 이미지를 담는 클래스
  • 어마무시하게 많이 사용되는 클래스
  • opencv에서 컬러는 CV_8UC3 으로 나타내고, 흑백은 CV_8UC1로 나타낸다.
    • 8은 8비트 즉 0~255 값으로 색정보를 담는다.
    • UC는 unsigned char를 나타내고,
    • 3 혹은 1은 8비트짜리가 3개 혹은 1개 있다는 것을 의미한다.
    • color는 BGR 세가지 정보를 위해 8비트가 세개 있어야 하고, gray scale은 8비트가 하나만 있어도 표현이 가능하다.

Open CV

open cv 설치에 대해 알아본다. 본문에서는 윈도우10과 비주얼 스투디오 2022를 사용하는 환경을 기준으로 한다.

설치

  • 설치 파일 다운로드

    • https://opencv.org/releases 에서 파일 다운로드
    • 이 글에서는 opencv 버전 4.5.5를 기준으로 한다.
    • opencv-4.5.5vc14_vc15.exe 파일 다운로드한다.

  • 설치 실행 파일 이용

    • opencv-4.5.5vc14_vc15.exe 파일을 실행하면 압축이 풀린다.
    • 압축을 풀 때 위치를 지정할 수 있다.
    • 설치는 압축을 푼것으로 완료.

  • opencv 파일 설명

    • opencv/build/include/opencv2 경로에 헤더파일들이 위치한다.
    • opencv/build/x64 아래 vc14 vc15 디렉토리가 있다.
    • vc14는 visual studio 2015 버전을 이용해 빌드한 파일들이 있고, vc15는 2017버전을 이용해 빌드한 파일들이 있다.
    • visual studio 2022 버전에서는 vc15를 사용한다.
    • vc15/bin 경로 아래에는 dll 파일이 있다.
    • vc15/lib 경로 아래에는 lib 파일이 있다.

  • 환경변수 등록

    • 설치한 opencv 를 사용하기 위해 환경 변수 등록을 해준다.
      • 변수 이름 OPENCV_DIR
      • 변수 값 opencv/build
    • path 항목 편집
      • %OPENCV_DIR%/x64/vc15/bin 항목 추가
    • cmd 에서 설치 확인
      • opencv_version 명령어를 입력하여 버전 정보를 확인 할 수 있으면 설치 완료

Visual Studio 2022 프로젝트 만들기

  • opencv 새 프로젝트 만들기

    • 새 프로젝트 만들기
    • 빈 프로젝트 선택
    • 솔루션 및 프로젝트를 같은 디렉토리에 배치

  • opencv 속성 추가

    • 헤더파일 위치 지정
      • 구성 속성 -> C/C++ -> 일반
      • 추가 포함 디렉터리 항목에 $(OPENCV_DIR)\include 입력
    • lib 폴더 위치 지정
      • 구성 속성 -> 링커 -> 일반
      • 추가 라이브러리 디렉터리 항목에 $(OPENCV_DIR)\x64\vc15\lib 입력
    • lib 파일 이름 지정
      • 구성 속성 -> 링커 -> 입력
      • 추가 종속성 항목에 opencv_world455d.dll 추가 (Release 모드는 opencv_world455.lib 추가)
