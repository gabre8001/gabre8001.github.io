Numpy мӢңмһ‘н•ҳкё°

python лӘЁл“Ҳмқё numpyлҘј м•Ңм•„ліҙкі  м„ нҳ•лҢҖмҲҳ м—°мӮ° кё°мҙҲлҘј м•Ңм•„ліёлӢӨ.

numpy лӘЁл“Ҳ мӮ¬мҡ©н•ҳкё°

мӢӨн–ү мӢңмјңліҙл©ҙ л„ҳнҢҢмқҙлҘј мӮ¬мҡ©н•ң л°°м—ҙ мқҙ лҚ” л№ лҘёкІғмқ„ нҷ•мқё н•  мҲҳ мһҲлӢӨ.

лӢӨм°Ёмӣҗ л°°м—ҙмқ„ м„ м–ён•  мҲҳ мһҲкі  shape мқ„ мқҙмҡ©н•ҙ л°°м—ҙмқҳ нҳ•нғңлҘј нҷ•мқён•  мҲҳлҸ„ мһҲлӢӨ.

numpy м—°мӮ°

  • лІЎн„°мҷҖ мҠӨм№јлқј м—°мӮ°
  • лІЎн„°мҷҖ лІЎн„° м—°мӮ°
  • к°Ғ лІЎн„°мқҳ мҡ”мҶҢ лҒјлҰ¬ м—°мӮ°

arrayмқҳ indexingкіј slicing

  • нҢҢмқҙмҚ¬мқҳ лҰ¬мҠӨнҠёмҷҖ мң мӮ¬н•ҳкІҢ м§„н–ү

Arrayмқҳ Broadcasting

  • н”јм—°мӮ°мһҗк°Җ м—°мӮ° к°ҖлҠҘн•ҳлҸ„лЎқ ліҖнҷҳмқ„ н•ҳм—¬ м—°мӮ°мқ„ м§„н–үн•ҳлҠ” кІғмқ„ Broadcasting мқҙлқј н•Ё.

  • M x N, M x 1

  • M x N, 1 x N
  • M x 1, 1 x N

мҳҒлІЎн„°(н–үл ¬)

  • мӣҗмҶҢк°Җ лӘЁл‘җ 0мқё лІЎн„°(н–үл ¬)

мқјлІЎн„°(н–үл ¬)

  • мӣҗмҶҢк°Җ лӘЁл‘җ 1мқё лІЎн„°(н–үл ¬)

лҢҖк°Ғн–үл ¬(diagonal matrix)

  • Main diagonalмқ„ м ңмҷён•ң м„ұл¶„мқҙ 0мқё н–үл ¬

н•ӯл“ұн–үл ¬(identity matrix)

  • Main diagonalмқҙ 1мқё лҢҖк°Ғн–үл ¬
  • мқёмһҗлЎң л“Өм–ҙк°ҖлҠ” dtypeмқҳ кё°ліё к°’мқҖ float

н–үл ¬кіұ(dot product)

  • н–үл ¬к°„мқҳ кіұм—°мӮ°

нҠёл ҲмқҙмҠӨ(trace)

  • main diagonalмқҳ н•©

н–үл ¬мӢқ(determinant)

  • н–үл ¬мқ„ лҢҖн‘ңн•ҳлҠ” к°’л“Ө мӨ‘ н•ҳлӮҳ

м—ӯн–үл ¬(inverse matrix)

  • н–үл ¬ Aм—җ лҢҖн•ҙ AB = BA = I лҘј л§ҢмЎұн•ҳлҠ” н–үл ¬ B = A^-1

кі мң к°’кіј кі мң лІЎн„°(eigenvalue and eigenvector)

  • м •л°©н–үл ¬ Aм—җ лҢҖн•ҙ $Ax = \lambda x$лҘј л§ҢмЎұн•ҳлҠ” мғҒмҲҳ$\lambda$мҷҖ мқҙм—җ лҢҖмқ‘н•ҳлҠ” лІЎн„°
  • 1м°Ёмӣҗ лІЎн„°мқҳ norm к°’мқ„ кө¬н•ҳлҠ” н•ЁмҲҳ мһ‘м„ұ
  • 2м°Ёмӣҗ лІЎн„°мқҳ singular matrix мқём§Җ нҷ•мқён•ҳлҠ” н•ЁмҲҳ мһ‘м„ұ

Vector Processor

лІЎн„° н”„лЎңм„ём„ң(vector processor or array processor)лҠ” лІЎн„°лқјкі  л¶ҲлҰ¬лҠ” лӢӨмҲҳмқҳ лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” лӘ…л №м–ҙлҘј к°Җм§„ CPUлҘј л§җн•ңлӢӨ. м»ҙн“Ён„°м—җм„ң лІЎн„°лһҖ 1м°Ёмӣҗ л°°м—ҙмқҳ лҚ°мқҙн„°лҘј лң»н•ңлӢӨ. лІЎн„° н”„лЎңм„ём„ңмҷҖ л°ҳлҢҖлҗҳлҠ” л§җлЎңлҠ” мҠӨм№јлқј н”„лЎңм„ём„ңк°Җ мһҲлҠ”лҚ° н•ң к°ңмқҳ лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” лӘ…л №м–ҙлҘј к°Җм§„ н”„лЎңм„ём„ңлҘј л§җн•ңлӢӨ. лҢҖл¶Җл¶„мқҳ CPUлҠ” мҠӨм№јлқј н”„лЎңм„ём„ңмқҙлӢӨ.

нҢҢмқҙн”„лқјмқё(pipeline)мқҙ ліөмҲҳмқҳ лӘ…л №м–ҙлҘј лҸҷмӢңм—җ мІҳлҰ¬н•ҳлҠ” кІғмқҙлқјл©ҙ лІЎн„° н”„лЎңм„ёмӢұмқҖ ліөмҲҳмқҳ лҚ°мқҙн„°лҘј лҸҷмӢңм—җ мІҳлҰ¬н•ҳлҠ” кІғмқҙлӢӨ. мҰү мҠӨм№јлқј к°’мқё 1к°ңмқҳ лҚ°мқҙн„°лҘј м—°мӮ°н•ҳлҠ” кІғмқҙ м•„лӢҲлқј лІЎн„°лЎң кө¬м„ұлҗң м—¬лҹ¬ лҚ°мқҙн„°лҘј 1к°ңмқҳ лӘ…л №м–ҙлЎң лІЎн„° н•©, м°Ё, кіұ л“ұмқҳ м—°мӮ°мқ„ н•ңлӢӨ. лІЎн„° м—°мӮ°мқ„мҶҢн”„нҠёмӣЁм–ҙк°Җ м•„лӢҢ н•ҳл“ңмӣЁм–ҙлЎң кі мҶҚмңјлЎң мІҳлҰ¬н•ңлӢӨ.

лІЎн„° н”„лЎңм„ём„ңлҠ” 1970л…„лҢҖ мІҳмқҢ лӮҳнғҖлӮ¬мңјл©° 1980л…„м—җм„ң 1990л…„лҢҖ лҸҷм•Ҳ мҠҲнҚј м»ҙн“Ён„°мқҳ кё°ліём Ғмқё нҳ•нғңмҳҖлӢӨ. мҠӨм№јлқј н”„лЎңм„ём„ң, нҠ№нһҲ л§ҲмқҙнҒ¬лЎңн”„лЎңм„ём„ңм—җм„ң м„ұлҠҘмқ„ лҶ’мқҙкё° мң„н•ҙ лІЎн„° н”„лЎңм„ём„ң кё°мҲ мқ„ лҸ„мһ…н•ң CPUк°Җ 1990л…„лҢҖ мҙҲ лӮҳнғҖлӮ¬лӢӨ. мҳӨлҠҳлӮ  лҢҖл¶Җл¶„мқҳ cpuлҠ” MMX, SSE, AltiVec к°ҷмқҙ лӢӨмҲҳмқҳ лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” лІ н„° н”„лЎңм„ёмӢұмқ„ мң„н•ң SIMD(Single Instruction Multiple Data) лӘ…л №м–ҙлҘј к°–м¶”кі  мһҲлӢӨ. лІЎн„° н”„лЎңм„ём„ң кё°мҲ мқҖ к·ёлһҳн”Ҫ к°ҖмҶҚкё°лӮҳ кІҢмһ„ мҪҳмҶ”м—җм„ңлҸ„ м°ҫм•„ ліј мҲҳ мһҲлӢӨ. 2000л…„ IBM, лҸ„мӢңл°”, мҶҢлӢҲк°Җ к°ңл°ңн•ң Cell ProcessorлҠ” 1к°ңмқҳ мҠӨм№јлқј н”„лЎңм„ём„ңмҷҖ 8к°ңмқҳ лІЎн„° н”„лЎңм„ём„ңлЎң кө¬м„ұлҗҳм–ҙ мһҲмңјл©° н”Ңл ҲмқҙмҠӨн…Ңмқҙм…ҳ 3м—җ мӮ¬мҡ©лҗҳм—ҲлӢӨ.

лІЎн„° н”„лЎңм„ёмӢұмқ„ мң„н•ң лҳҗ лӢӨлҘё CPU л””мһҗмқёмңјлЎң лӢӨмҲҳмқҳ лӘ…л №м–ҙлЎң лӢӨмҲҳмқҳ лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” MIMD (Multiple Instruction Multiple Data)к°Җ мһҲм§Җл§Ң м „л¬ём Ғмқё мҡ©лҸ„лЎңл§Ң мӮ¬мҡ©лҗ  лҝҗ мқјл°ҳм Ғмқё мҡ©лҸ„лЎң м“°мқҙм§ҖлҠ” м•Ҡм•ҳлӢӨ.

м—ӯмӮ¬

мөңмҙҲлЎң лҸҷмһ‘лҗҳлҠ” лІЎн„° н”„лЎңм„ём„ңлҠ” 1960л…„лҢҖ мҙҲ, мӣЁмҠӨнҢ…н•ҳмҡ°мҠӨмқҳ мҶ”лЎңлӘ¬ н”„лЎңм қнҠёмҳҖлӢӨ. мҶ”лЎңлӘ¬мқҳ лӘ©лЈҢлҠ” л§ҺмқҖ мҲҳмқҳ лӢЁмҲңн•ң мҲҳм№ҳ м—°мӮ° мҪ”н”„лЎңм„ём„ң(ALU)лҘј 1к°ңмқҳ л§ҲмҠӨн„° CPUлЎң м ңм–ҙн•ҳлҠ” л°©лІ•мқ„ мӮ¬мҡ©н•ҙ мҲҳм№ҳ м—°мӮ° м„ұлҠҘмқ„ к·№м ҒмңјлЎң лҒҢм–ҙмҳ¬лҰ¬лҠ” кІғмқҙм—ҲлӢӨ. CPUлҠ” мӮ¬мқҙнҒҙ лӢ№ н•ҳлӮҳмқҳ кіөнҶө лӘ…л №м–ҙлҘј лӘЁл“  ALUлЎң ліҙлғҲм§Җл§Ң лҚ°мқҙн„°лҠ” ALU л§ҲлӢӨ к°Ғк°Ғ лӢӨлҘҙкІҢ ліҙлӮҙмЈјм—ҲлӢӨ. мқҙ л°©лІ•мқҖ мҶ”лЎңлӘ¬ лЁёмӢ м—җм„ң н•ҳлӮҳмқҳ м•Ңкі лҰ¬мҰҳмқ„ л°°м—ҙ нҳ•нғңмқҳ л§ҺмқҖ лҚ°мқҙн„°м—җ мқҙмҡ©н•  мҲҳ мһҲлҸ„лЎқ н•ҙмӨҖлӢӨ. 1962л…„ мӣЁмҠӨнҢ…н•ҳмҡ°мҠӨлҠ” мҶ”лЎңлӘ¬ н”„лЎңм қнҠёлҘј мӨ‘лӢЁн•ҳмҳҖм§Җл§Ң лІЎн„° н”„лЎңм„ём„ңмқҳ к°ңл°ңм—җ лҢҖн•ң л…ёл ҘмқҖ мқјлҰ¬л…ёмқҙмҰҲ лҢҖн•ҷмқҳ мқјлҰ¬м•… IVм—җм„ң лӢӨмӢң мӢңмһ‘лҗҳм—ҲлӢӨ. мқјлҰ¬м•…мқҖ 256к°ңмқҳ ALUлҘј мӮ¬мҡ©н•ҙ 1 GFLOPSмқҳ лЁёмӢ мқ„ л§Ңл“Өкі мһҗ н•ҳмҳҖмңјлӮҳ 1972л…„ мҷ„м„ұлҗҳм—Ҳмқ„ л•ҢлҠ” 64к°ңмқҳ ALUлЎң 100 ~ 150 MFLOPSмқҳ м„ұлҠҘл°–м—җ лӮҙм§Җ лӘ»н–ҲлӢӨ. к·ёл Үм§Җл§Ң кё°ліё к°ңл…җмқҖ м¶©мӢӨн•ҙм„ң мң мІҙм—ӯн•ҷкіј к°ҷмқҖ лҚ°мқҙн„° мӨ‘мӢ¬мқҳ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ңлҠ” м„ёкі„м—җм„ң к°ҖмһҘ л№ лҘё лЁёмӢ мқҙм—ҲлӢӨ. мқјлҰ¬м•…мқҳ к°Ғ лҚ°мқҙн„° мҡ”мҶҢл§ҲлӢӨ лі„лҸ„мқҳ ALUлҘј мӮ¬мҡ©н•ң л°©лІ•мқҖ мқҙнӣ„ лІЎн„° н”„лЎңм„ём„ң м„Өкі„м—җм„ңлҸ„ мқјл°ҳм Ғмқё кІғмқҖ м•„лӢҲм—Ҳмңјл©° лҢҖк·ңлӘЁ лі‘л ¬ м»ҙн“ЁнҢ… лІ”мЈјлЎң кө¬л¶„лҗҳкё°лҸ„ н•ңлӢӨ.

мҙҲм°Ҫкё° лІЎн„° н”„лЎңм„ём„ң мӢңмҠӨн…ң мӨ‘ мң лӘ…н•ң кІғмқҖ CDC(Control Data Corporation)мқҳ STAR-100кіј н…ҚмӮ¬мҠӨ мқёмҠӨнҠёлЈЁлЁјмё мқҳ ASC(Advanced Scientific Computer)мҳҖлӢӨ. ASCмқҳ ALUлҠ” 1к°ңмқҳ нҢҢмқҙн”„лҘј к°Җм§„ нҢҢмқҙн”„лқјмқё м•„нӮӨн…ҚмІҳлҘј мӮ¬мҡ©н•ҙ мҠӨм№јлқјмҷҖ лІЎн„° м—°мӮ°мқ„ лӘЁл‘җ мІҳлҰ¬н•  мҲҳ мһҲм—Ҳмңјл©° мөңлҢҖ м„ұлҠҘмқҖ кёёмқҙк°Җ кёҙ лІЎн„°лҘј мІҳлҰ¬н•  кІҪмҡ° м•Ҫ 20MFLOPSм—җ лӢ¬н–ҲлӢӨ. нҷ•мһҘ ALU кө¬м„ұм—җм„ңлҠ” 2к°ңлӮҳ 4к°ңлЎң нҢҢмқҙн”„лҘј нҷ•мһҘн•ҙ 2л°°лӮҳ 4л°°мқҳ м„ұлҠҘмқ„ м–»мқ„ мҲҳ мһҲм—ҲлӢӨ. л©”лӘЁлҰ¬ лҢҖм—ӯнҸӯмқҖ нҷ•мһҘ лӘЁл“ңлҘј мӮ¬мҡ©н•ҳлҠ”лҚ° м¶©л¶„н–ҲлӢӨ. STARлҠ” CDC 7600 к°ҷмқҖ CDCмқҳ мҠҲнҚј м»ҙн“Ён„°ліҙлӢӨлҠҗ г„ҙлҠҗлҰҙм§ҖлҠ” лӘ°лқјлҸ„ лҚ°мқҙн„° кҙҖл Ё мһ‘м—…м—җм„ңлҠ” нҒ¬кё°лҸ„ мһ‘кі  к°ҖкІ©лҸ„ лӮ®м•ҳлӢӨ. н•ҳм§Җл§Ң STARлҠ” лІЎн„° лӘ…л №м–ҙлҘј л””мҪ”л”©н•ҳкі  н”„лЎңм„ёмҠӨмқҳ мӢӨн–үмқ„ мӨҖл№„н•ҳлҠ”лҚ° мӢңк°„мқҙ л§Һмқҙ кұёл ёлӢӨ. к·ёлһҳм„ң мӢӨм ң м„ұлҠҘмқ„ л°ңнңҳн•ҳл Өл©ҙ л§Өмҡ° мқјм •н•ң лҚ°мқҙн„° м§‘н•©мқҙ н•„мҡ”н–ҲлӢӨ.

лІЎн„° кё°мҲ мқҖ нҒ¬л Ҳмқҙ-1м—җм„ң мІҳмқҢмңјлЎң мҷ„м„ұлҗҳм—ҲлӢӨ. нҒ¬л ҲмқҙлҠ” STARлӮҳ ASC к°ҷмқҙ лҚ°мқҙн„°лҘј л©”лӘЁлҰ¬м—җ лӮЁкІЁл‘җм§Җ м•Ҡкі  8к°ңмқҳ лІЎн„° л Ҳм§ҖмҠӨн„°лҘј мӮ¬мҡ©н–ҲлӢӨ. лІЎн„° л Ҳм§ҖмҠӨн„°лҠ” 64л№„нҠё мӣҢл“ң 64к°ңлЎң мқҙлЈЁм–ҙм ё мһҲлӢӨ. лІЎн„° лӘ…л №м–ҙлҠ” л Ҳм§ҖмҠӨн„° мӮ¬мқҙм—җм„ң мӢӨн–үлҗҳм—ҲлҠ”лҚ° л©”мқё л©”лӘЁлҰ¬м—җм„ң к°Җм ёмҳӨлҠ” л°©лІ•ліҙлӢӨ нӣЁм”¬ л№ЁлһҗлӢӨ. нҒ¬л ҲмқҙлҠ” лІЎн„° лӘ…л №м–ҙлҘј мӢӨн–үн•ҳлҠ”лҚ° лӢӨмҲҳмқҳ ALUлҘј мӮ¬мҡ©н•ҳлҠ” лҢҖмӢ  нҢҢмқҙн”„лқјмқё лі‘л №нҷ”лҘј мӮ¬мҡ©н–ҲлӢӨ. мқҙ л°©лІ•мқҖ лӘ…л №м–ҙм—җ л”°лқј нҢҢмқҙн”„лқјмқёмқ„ лӮҳлҲ„м–ҙ лҶ“мқҖ кІғмңјлЎң мҳҲлҘј л“Өм–ҙ лҚ§м…Ҳ/лә„м…ҲмқҖ кіұм…ҲкіјлҠ” лӢӨлҘё нҢҢмқҙн”„лқјмқём—җм„ң мӢӨн–үлҗңлӢӨ. мқҙлҹ° кё°мҲ мқ„ лІЎн„° мІҙмқҙлӢқмқҙлқј л¶ҖлҘёлӢӨ. нҒ¬л Ҳмқҙ-1мқҳ мқјл°ҳм Ғмқё м„ұлҠҘмқҖ 80 MFLOPS м Җмҳ«мҳҖмңјлӮҳ 3к°ңмқҳ мІҙмқёмңјлЎң мӢӨн–үн•  кІҪмҡ° мөңлҢҖ м„ұлҠҘмқҖ 240 MFLOPSлҘј кё°лЎқн–ҲлӢӨ.

лІЎн„° н”„лЎңм„ёмӢұ кё°мҲ мқҖ мөңк·ј кұ°мқҳ лӘЁл“  CPUм—җм„ң SIMDлқјлҠ” лӘ…м№ӯмңјлЎң м¶”к°Җлҗҳм—ҲлӢӨ. лІ н„° мң лӢӣмқҖ н”„лЎңк·ёлһЁмңјлЎңл¶Җн„° мІҳлҰ¬н•  лҚ°мқҙн„°лҘј л°ӣм•„ л©”мқё мҠӨм№јлқј CPUмҷҖ лҸҷмӢңм—җ мһ‘лҸҷн•ңлӢӨ.
