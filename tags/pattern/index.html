Proxy Pattern

Proxy Pattern

м–ҙл–Ө к°қмІҙм—җ лҢҖн•ң м ‘к·јмқ„ м ңм–ҙн•ҳлҠ” мҡ©лҸ„лЎң лҢҖлҰ¬мқёмқҙлӮҳ лҢҖліҖмқём—җ н•ҙлӢ№н•ҳлҠ” к°қмІҙлҘј м ңкіөн•ҳлҠ” нҢЁн„ҙ

  1. virtual proxy
    нҒҙлһҳмҠӨк°Җ мӮ¬мҡ©н•ҳлҠ” мһҗмӣҗмқҙ л§Һмқ„ кІҪмҡ° мһ‘м—…мқҙ н•„мҡ”н•  л•Ңм—җл§Ң нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ҳлҸ„лЎқ н”„лЎқмӢңлҘј мӮ¬мҡ©
  2. Protection proxy
    нҒҙлһҳмҠӨм—җ лҢҖн•ң м ‘к·јмқ„ м ңм–ҙн•ҳкё° мң„н•ң кІҪмҡ° н”„лЎқмӢң нҒҙлһҳмҠӨм—җм„ң мЈјмІҙ нҒҙлһҳмҠӨм—җ лҢҖн•ң м ‘к·јк¶Ңн•ңмқ„ нҷ•мқён•ҳм—¬ л©”мҶҢл“ң нҳём¶ңмқ„ м ңм–ҙ
  3. remote proxy
    н”„лЎқмӢң нҒҙлһҳмҠӨлҠ” лЎңм»¬м—җ мһҲкі  мЈјмІҙ нҒҙлһҳмҠӨлҠ” remoteлЎң мЎҙмһ¬н•ҳлҠ” кІҪмҡ° мӮ¬мҡ©

кө¬нҳ„ мҳҲмӢң

м•„мЈј к°„лӢЁн•ң мҳҲм ң н•ҳлӮҳл§Ң л“Өм–ҙліҙлҸ„лЎқ н•ңлӢӨ. н”„лЎқмӢңмқҳ кө¬нҳ„м—җм„ң н”„лЎқмӢң л©”мҶҢл“ңм—җ ліё м„ңл№„мҠӨ к°қмІҙлҘј мғқм„ұн•ҳлҠ” кІҪмҡ° virtual proxy мқҙкі , м ‘к·ј к¶Ңн•ңмқ„ нҷ•мқён•ҳм—¬ ліё м„ңл№„мҠӨмқҳ л©”мҶҢл“ң нҳём¶ңмқ„ м ңм–ҙн•ҳлҠ” кІҪмҡ° protection proxy, лҰ¬лӘЁнҠё нҶөмӢ мқ„ нҶөн•ҙ ліё м„ңл№„мҠӨлҘј нҳём¶ңн•ҳлҠ” кІҪмҡ° remote proxyлқј н•ңлӢӨ. м•„лһҳлҠ” м„ңл№„мҠӨк°Җ мһҲкі  мқҙ м„ңл№„мҠӨлҘј нҳём¶ңн•ҳкё° мң„н•ң н”„лЎқмӢңк°Җ мЎҙмһ¬н•ҳлҠ” кІҪмҡ°мқҙлӢӨ.

л©”мқё н•ЁмҲҳм—җм„ң н”„лЎқмӢңлҘј нҳём¶ңн•ҳм—¬ кІ°кіјм ҒмңјлЎң м„ңл№„мҠӨ кө¬нҳ„мІҙлҘј нҳём¶ңн•ҳлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ.

л””мһҗмқё нҢЁн„ҙ

  • мғқм„ұ creational нҢЁн„ҙмқҖ к°қмІҙ мғқм„ұм—җ кҙҖл Ёлҗң нҢЁн„ҙ, к°қмІҙмқҳ мғқм„ұкіј мЎ°н•©мқ„ мәЎмҠҗнҷ”н•ҙ нҠ№м • к°қмІҙк°Җ мғқм„ұлҗҳкұ°лӮҳ ліҖкІҪлҗҳм–ҙлҸ„ н”„лЎңк·ёлһЁ кө¬мЎ°м—җ мҳҒн–Ҙмқ„ нҒ¬кІҢ л°ӣм§Җ м•ҠлҸ„лЎқ мң м—°м„ұмқ„ м ңкіөн•ңлӢӨ.

Observer Pattern

Observer Pattern

  • н•ң к°қмІҙ мғҒнғң ліҖнҷ”м—җ лӢӨлқј лӢӨлҘё к°қмІҙ мғҒнғңлҸ„ м—°лҸҷлҗҳлҸ„лЎқ мқјлҢҖлӢӨ к°қмІҙ мқҳмЎҙ кҙҖкі„лҘј кө¬м„ұн•ҳлҠ” нҢЁн„ҙ
  • лҚ°мқҙн„° ліҖкІҪмқҙ л°ңмғқн–Ҳмқ„ кІ°мҡ° мғҒлҢҖ нҒҙлһҳмҠӨлӮҳ к°қмІҙм—җ мқҳмЎҙн•ҳм§Җ м•Ҡмңјл©ҙм„ң лҚ°мқҙн„° ліҖкІҪмқ„ нҶөліҙн•ҳкі мһҗ н•  л•Ң мң мҡ©
  • мҳөм ҖлІ„ нҢЁн„ҙмқҖ нҶөліҙ лҢҖмғҒ к°қмІҙмқҳ кҙҖлҰ¬лҘҙ Subject нҒҙлһҳмҠӨмҷҖ Observer мқён„°нҺҳмқҙмҠӨлЎң мқјл°ҳнҷ”
  • лҚ°мқҙн„° ліҖкІҪмқ„ нҶөліҙн•ҳлҠ” нҒҙлһҳмҠӨ(CondreteSubject)лҠ” нҶөліҙ лҢҖмғҒ нҒҙлһҳмҠӨлӮҳ к°қмІҙм—җ лҢҖн•ң мқҳмЎҙм„ұмқ„ м—Ҷм•Ё мҲҳ мһҲмқҢ

кө¬нҳ„ мҳҲмӢң

м•јкө¬ м№ҙмҡҙнҠё м •ліҙм—җ лҢҖн•ң м •ліҙлҘј ліҙм—¬мЈјлҠ” мӢңмҠӨн…ңмқҙ мһҲлӢӨкі  к°Җм •н•ҳмһҗ. м•јкө¬ м№ҙмҡҙнҠём—җ лҢҖн•ң м •ліҙлҘј к°Җм§„ Subjectк°Җ мЎҙмһ¬н•ҳкі  мқҙлҘј м—¬лҹ¬кө°лҚ°м—җм„ң ліҙм—¬мЈјлҠ” Observer кө¬нҳ„мІҙл“Өмқҙ мһҲлӢӨ. м •ліҙлҠ” н•ңкө°лҚ°м—җм„ң м ңкіөн•ҳкі  Observer кө¬нҳ„мІҙл“ӨмқҖ м—¬лҹ¬ кіім—җм„ң м—¬лҹ¬ л°©мӢқмңјлЎң м •ліҙлҘј ліҙм—¬мӨ„ мҲҳ мһҲлӢӨ. Observer кө¬нҳ„мІҙл“ӨмқҖ м—¬лҹ¬к°Җм§Җ л°©лІ•мңјлЎң кө¬нҳ„ лҗ  мҲҳ мһҲкі  мқҙм—җ л”°лқј м •ліҙлҘј ліҙм—¬мЈјлҠ” л°©лІ•лҸ„ м—¬лҹ¬к°Җм§Җк°Җ лҗ  мҲҳ мһҲлӢӨ. Observer кө¬нҳ„мІҙк°Җ м¶”к°Җ лҗҳкұ°лӮҳ лӢӨлҘё л°©мӢқмқ„ кө¬нҳ„ лҗҳлҚ”лқјлҸ„ м •ліҙлҘј м ңкіөн•ҳлҠ” SubjectлҠ” м •ліҙлҘј м ңкіөн•ҳкё°л§Ң н•ҳкё° л•Ңл¬ём—җ мқҙм—җ л”°лҘё ліҖкІҪмқҙ н•„мҡ” м—ҶлӢӨ.

мң„лҠ” м•јкө¬ м •ліҙлҘј м ңкіөн•ҳлҠ” Subject BaseballStatusм—җ лҢҖн•ң мҪ”л“ң

мң„лҠ” м •ліҙлҘј л°ӣм•„ ліҙм—¬мЈјлҠ” Observer кө¬нҳ„мІҙмқҙлӢӨ. мқҙ кө¬нҳ„мІҙлҠ” лӢӨлҘё л°©мӢқмқ„ кө¬нҳ„лҗң нҒҙлһҳмҠӨк°Җ м¶”к°Җ лҗ  мҲҳ мһҲмңјлӮҳ м •ліҙлҘј м ңкіөн•ҳлҠ” BaseballStatusмқё SubjectлҠ” ліҖкІҪмқҙ н•„мҡ” м—ҶлӢӨ.


BaseballStatusлҘј observer кө¬нҳ„мІҙл“Өм—җ л“ұлЎқн•ҳкі  setGameмқ„ нҶөн•ҙ BaseballStatusк°Җ ліҖкІҪлҗ  л•Ңл§ҲлӢӨ м •ліҙк°Җ н‘ңмӢңлҗҳлҠ” кІғмқ„ нҷ•мқё н•  мҲҳ мһҲлӢӨ.
