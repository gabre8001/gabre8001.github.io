자율주행 자동차 기술 소개

자율주행 프로세스

  • 인지

    • 경로탐색, 차량/인프라 통신, 센서
    • 고정지물 인식 및 경로탐색, 변동지물 및 이동물 인식

  • 판단

    • 주행상황 판단 및 주행전략 결정
    • 주행경로 생성

  • 제어

    • 차량제어
    • 엔진 가감속, 조향

요소 기술

  • 고정밀 지도

    • 기존 네비게이션 지도보다 정밀한 지도
    • 100 ~ 200 mm 정밀도 제공
    • 차선정보, 가드레일, 도로 곡률, 결사, 신호등 위치, 교통표식과 같은 부가정보 포함
    • 벡터 맵
    • 포인트 맵

  • Localization 기술

    • 정밀지도와 차량의 센서와 연동하여 차량의 현재 위치 파악

  • Global Path Planning (Route Planning)

    • 목적지가지의 경로 찾기
    • 중간목적지 또는 최종목적지까지의 경로
    • 교차로에서 행동 결정

  • Object Detection

    • 주변 차량, 보행자, 오토바이, 자전거 등을 인식

  • Object Tracking

    • 각 객체에 고유 ID 부여하여 추적, 예상되는 주행경로 예측

  • Local Path Planning (Trajectory Planning)

    • 다음 이동할 곳 경로 찾기(충돌 회피 교려)
    • 여러개의 후보 경로 확보
    • 끊임 없이 후보 삭제, 신규후보 등록 작업을 반복
    • 실시간성이 중요 -> 시스템 최적화 필요

  • Behavior Selector

    • 행위 결정
      • 운전 의사결정
      • 운전 방법, 성향

  • Local Path Following (Trajectory Following)

    • 경로 따라 차량을 운전하기
    • Pure pursuit 알고리즘
      • 차량의 현재 위치를 결정
      • 차향에서 가장 가까운 경로상의 점을 찾는다.
      • 목표점을 찾는다.
      • 곡률을 계산하고 해당 곡률로 차량의 방향을 업데이트 해준다.
      • 차량의 위치를 업데이트 한다.

  • Vehicle Control

    • 주행 제어
      • 원하는 대로 차량을 움직이게 만들기
    • 단순하지 않음
      • 차량 운동학
      • 관성, 마찰력, 미끄러짐 등 있음

자율주행 통합 플랫폼

    • https://www.autoware.org

    • SAE-레벨2

    • 30개 이상의 국가에서 사용
    • 100개 이상의 회사들에 사용
    • 20개 이상의 차량 모델에 탑재

자율주행 자동차 구현기술

  • 자율주행 알고리즘
    • 센싱 - 주변정보 획득
    • 인지 - Localization, Object Detection, Object Tracking
    • 의사결정 - 동작 예측, 경로 계획, 장애물 회피
  • 자율주행 클라이언트 시스템
    • 소프트웨어
      • 실시간성과 신뢰성 확보
      • ROS 문제점 해결 필요. master 장애시 대책 필요. 메시지 브로드캐스팅시 성능 저하. 노드 해킹 위움.
    • 하드웨어
      • 성능 향상. 파이프라인 병령 프로세싱. 가성비. 전성비. 발열제어.
  • 자율주행 클라우드 플랫폼
    • 분산 컴퓨팅
      • 시뮬레이션
      • HD맵 생성
    • 분산 스토리지
      • 팁러닝 모델 학습. 학습 데이터 준비. 학습진행. 모델의 유효성과 효율성 개선.
