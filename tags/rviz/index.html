RVIZ로 자동차 움직여 보기

ROS, RVIZ를 이용해 자동차를 8자로 움직여 본다.

  • 본문은 정확한 자동차 역학에 기반한 자동차 구동을 구현한 것은 아님을 밝힌다.
  • 일단 눈으로 보기에 8자와 유사하게 움직이는 것을 해보는데 주안점을 두었다.
  • 더 정확한 구현은 오도메트리에 대한 설명과 함께 나중에 다루도록 한다.

패키지 준비하기

  • 자동차 움직여 보기를 위한 ROS 패키지를 작성한다.
  • ros 워크 스페이스 내 src 위치에 가서 아래와 같이 명령어를 실행한다.
  • rviz_xycar라는 디렉토리가 생성되면서 이후 나열된 패키지들이 의존성에 추가된다.
  • 가장 마지막에 있는 message_generation은 사용자가 직접 만든 메시지를 사용하기 위해 필요한 의존성으로 이에 대한 상세한 방법은 아래 링크를 참조하도록 한다.
    • http://wiki.ros.org/ROS/Tutorials/CreatingMsgAndSrv
    • 이 과정에서 메시지 빌드에 실패할 경우 {워크스페이스}/devel/setup.bash 를 실행해주거나 정 안되면 재부팅을 해준다.

RVIZ

RVIZ와 URDF에 대해 알아본다.

RVIZ

  • ROS와 연동하여 사용하는 3D 시각화 툴.
  • RVIZ 구동을 위해서는 아래와 같이 런치 파일 편집을 통해 ROS 실행을 하면 확인 가능하다.
    • pkg를 rviz로, type도 rviz로 생성한 노드를 통해 RVIZ를 실행한다.
    • 3D 시각화를 위해 필요한 파일이 urdf 파일이다.

URDF

  • Unified Robot Description Format

  • 로봇을 구현하는데 쓰이는 하나의 포맷으로 RVIZ에서 시각화를 통해 확인 할 수 있다.

  • 좌표계

    • 위치나 크기를 표현하기 우해 데카르트 좌표계 x,y,z를 사용
    • 회전을 표현하기 위해 오일러 각도 roll, pitch, yaw를 사용
    • 사용 단위
      • 길이: 미터 m
      • 각도: 라디안 radian
      • 질량: 킬로그램 kg
      • 속도: m/s (병진/직진운동)
      • 각속도: radian/s (회전운동)

  • 형상 표현

    • cylinder: 원통
    • box: 상자
    • sphere: 공

  • 색상 표현

    • RGB 3원색과 투과율 A를 0~1사이의 숫자로 정의함

  • 기구의 표현

    • base: 고정 부분(grounded)
    • link: 관절에 연결되는 로봇 팔의 부분
    • joint: 링크를 연결하는 부위로 보통 모터의 회전을 통해 움직임을 만듬

좌표계

  • 위치나 크기를 표현하기 우해 데카르트 좌표계 x,y,z를 사용
  • 회전을 표현하기 위해 오일러 각도 roll, pitch, yaw를 사용

단위

  • 길이: 미터 m
  • 각도: 라디안 radian
  • 질량: 킬로그램 kg
  • 속도: m/s (병진/직진운동)
  • 각속도: radian/s (회전운동)

형상 표현

  • cylinder: 원통
  • box: 상자
  • sphere: 공

색상 표현

  • RGB 3원색과 투과율 A를 0~1사이의 숫자로 정의함

기구의 표현

  • base: 고정 부분(grounded)

  • link: 관절에 연결되는 로봇 팔의 부분

    • visual: 시각화를 위해 형상과 위치를 정의
      • geometry: 형상 정의 원동, 상자, 공
      • origin: 고정축을 기준으로 link 형상의 roll, pitch, yaw 위치를 라디안으로 나타내고 x,y,z 좌표 위치를 미터 단위로 지정
      • material: 현상의 컬러값 지정

  • joint: 링크를 연결하는 부위로 보통 모터의 회전을 통해 움직임을 만듬

    • parent: parent frame의 이름을 지정해서 child frame과 연결
    • child: child frame의 이름을 지정해서 parent frame과 연결
    • origin: 고정축을 기준으로 형상의 r,p,y 위치를 라디안으로 나타내고, x,y,z 좌표 위치를 미터 단위로 지정.
    • joint는 parent의 origin, child는 joint의 origin을 고정축으로 정함
    • limit: joint의 운동범위 제한값을 지정
      • lower: revolution type의 joint에서 최저 각을 지정
      • upper: revolution type의 joint에서 최대 각을 지정
      • effort: N 힘의 최대값을 지정
      • velocity: radian/s 최대 속도를 지정

joint의 동작 정의

  • fixed: 고정
  • revolution: 작동 범위 제한
  • continuous: 연속 회전
